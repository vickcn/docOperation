# 文檔轉換工具

這是一個基於 Python 的文檔轉換工具，可以處理 .docx 文件並進行和弦轉調。

## 功能

- 支持 .docx 文件的解析和重建
- 支持和弦轉調
- 保留原始文檔格式
- 支持中文和特殊字符

## 本地運行

1. 克隆倉庫：
```bash
git clone https://github.com/vickcn/docOperation.git
cd docOperation
```

2. 安裝依賴：
```bash
pip install -r requirements.txt
```

3. 運行應用：
```bash
python app.py
```

4. 訪問 http://localhost:5418

## 技術棧

- Python
- Flask
- python-docx
